📋 GreytHR API Setup Guide

Step 1 — Enable API Access
→ Login to GreytHR as Super Admin → Settings → API & Integration → API Access
→ Generate a new API Token → copy it and paste above

Step 2 — Find Your Subdomain
→ Your GreytHR URL is https://[subdomain].greythr.com
→ Enter only the subdomain part (e.g. decisionfoundry) above

Step 3 — Permissions Required
→ The API token must have: Employee Read, Organisation Read

Step 4 — Test & Sync
→ Click Test Connection to verify credentials
→ Click Sync Now to import all active employees
→ Synced employees get accounts automatically — they can log in with their @decisionfoundry.com email

ℹ️ What gets synced: Full name, email, department, designation, employee ID, date of joining, date of birth, reporting manager, employment status
